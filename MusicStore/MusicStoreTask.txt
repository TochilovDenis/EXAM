Создать приложение «Музыкальный магазин».  
Основная задача проекта: учитывать текущий ассортимент музыкальных пластинок в магазине. 

1. Необходимо хранить следующую информацию о пластинках: 
- название пластинки 
- название коллектива 
- название издателя 
- количество треков 
- жанр
- год издания
- себестоимость
- цена для продажи.

2. Приложение должно позволять:
- добавлять пластинки 
- удалять пластинки
- редактировать параметры пластинок
- продавать пластинки
- списывать пластинки вносить пластинки в акции (например, неделя пластинок джаза со скидкой 10%) откладывать диски для конкретного покупателя. 

3. Приложение должно предоставить функциональность по поиску дисков по таким параметрам:
- название диска,
- исполнитель
- жанр

4. Приложение должно предоставлять возможность просмотреть:
- список новинок
- список самых продаваемых пластинок
- список самых популярных авторов
- список самых популярных жанров по итогам дня, недели, месяца, года.
 
5. Необходимо предусмотреть возможность входа по логину и паролю.

6. Также нужно сделать возможность регистрации постоянных покупателей и создать систему скидок в зависимости от накопленной суммы потраченных средств.
